🎬 BidCraft – User Manual for VFX Supervisors & Producers

🧠 What Is BidCraft?

BidCraft is a no-install, browser-based tool for VFX teams. It takes raw screenplay text and transforms it into a properly formatted breakdown layout, ready for tagging, exporting, and further scheduling work.

It's ideal for:

  • Script-to-bid workflows
  • First-pass VFX scene tagging
  • Early creative/technical planning
  • Deliverables estimation

No login. No software install. Fully offline.

🖥️ How It Works – Overview

You paste in a raw script or scene dump. BidCraft:

  • Reformats it into screenplay structure
  • Highlights scene headings (INT/EXT etc.)
  • Cleans spacing & removes noise
  • Bold-tags character names & visual cues
  • (Optionally) allows VFX tagging per scene
  • Outputs a clean, styled preview ready for export or copy

📂 File Setup

Open the HTML file Project_BidCraft_YYYYMMDD_V00.html

Runs in your browser locally (Chrome, Safari, Firefox)

📝 Step-by-Step Breakdown Workflow

  1. Open BidCraft
    Open the .html file in a browser
    The interface loads instantly — no installation
  2. Paste Your Script
    Use a raw .txt, .pdf, or export from Final Draft
    Paste into the main textarea
    Click the Format button
  3. What Happens Behind the Scenes
    BidCraft automatically:
    • 🧹 Removes page/scene numbers
    • 🔍 Detects scene headings
    • 🎭 Character names in ALL CAPS are centered + bolded
    • 🔠 ALL-CAPS words are bolded
    • 🧼 Spacing cleaned up

🎯 BidCraft – VFX Tagging Workflow

  1. Select Text to Tag
    In the formatted script viewer, highlight any text that relates to a visual effect.
    Example: "A CG alien leaps through the fire"
  2. Assign Scene & Shot Numbers
    A small tagging panel appears. Input the:
    • Scene number (e.g., 27)
    • Shot number or code (e.g., 27A, 27B, etc.)
  3. Select VFX Tags
    Choose one or more tags from a checklist (e.g.):
    • CG Character
    • Set Extension
    • Green Screen
    • Fire FX
    • Crowd Duplication
  4. Add Notes
    Add any additional detail that may help vendors or supervisors later:
    (e.g., "Creature enters frame from behind wreckage. Full CG.")
  5. Manage Custom Tags
    Click "Manage Tags" to:
    • Create your own custom tags
    • Edit existing ones
    • Delete unused or outdated tags
  6. View Tag History
    Click "Tagged Shots" to:
    • See all your tagged moments
    • Filter by tag type, scene, or text
    • Edit or delete existing entries
  7. Export to CSV
    When ready, export your breakdown as a .csv file
    This file can be imported into:
    • Excel
    • Google Sheets
    • Budgeting tools like ShotGrid or Movie Magic Budgeting
    Ideal for creating vendor bid packages or early quoting docs

🛠 Tips & Best Practices

  • Use plain-text scripts (copy-pasted from PDF or exported from Final Draft)
  • Check the formatting after pasting — re-run "Format" if needed
  • Works well with both features and episodic scripts
  • If needed, clean page numbers or formatting manually before pasting
  • Keep it offline — your data never leaves your machine
